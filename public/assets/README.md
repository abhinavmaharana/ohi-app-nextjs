# Assets Folder

Please copy the `assets` folder from the original website here.

The folder structure should be:
```
public/
  assets/
    img/
      logo.png
      herophone.png
      banner.png
      phone-pricing.png
      insta.png
```

You can copy it using:
```bash
cp -r /path/to/Ohi-website-2.0-main/assets ./public/
```
